Laços e Condicionais em JavaScript

Condicional (if, else)

Este exemplo verifica se um número é maior que outro. Você pode inserir um número e um valor para comparar.

if (número > valor) {
              resultado = "O número é maior.";
          } else {
              resultado = "O número é menor ou igual.";
          }

Condicional (if, else if, else)

Em lógica de programação, uma estrutura condicional permite que diferentes blocos de código sejam executados dependendo de uma condição. O exemplo abaixo verifica se um número é positivo, negativo ou zero. A instrução if verifica a primeira condição (se o número é maior que zero), a instrução else if verifica a segunda condição (se o número é menor que zero), e a instrução else é executada se nenhuma das condições anteriores for verdadeira (ou seja, o número é zero). 

if (número > 0) {
          resultado = "positivo";
      } else if (número < 0) {
          resultado = "negativo";
      } else {
          resultado = "zero";
      }

Laço For

O laço for é uma estrutura de repetição que permite que um bloco de código seja executado várias vezes. É comumente usado quando o número de repetições é conhecido. No exemplo abaixo, o laço for conta de 1 até o número que você escolher. A variável i é inicializada com 1, e o laço continua enquanto i for menor ou igual ao número inserido, incrementando i a cada repetição. 

for (var i = 1; i <= número; i++) {
      resultado += i + " ";
  }

Laço While

O laço while é outra estrutura de repetição que executa um bloco de código enquanto uma condição for verdadeira. É comumente usado quando o número de repetições não é conhecido antecipadamente. No exemplo abaixo, o laço while conta de um número que você escolher até 1. A condição verifica se i é maior que 0, e o laço continua enquanto essa condição for verdadeira, decrementando i a cada repetição.

var i = número;
        while (i > 0) {
          resultado += i + " ";
          i--;
      }

Switch Case

O switch é uma estrutura de controle que permite que um bloco de código seja executado de várias maneiras diferentes, dependendo do valor de uma expressão. É uma alternativa mais limpa e eficiente para uma série de instruções if else if quando você precisa comparar uma variável com vários valores literais. No exemplo abaixo, o valor inserido é comparado com números de 1 a 7, representando os dias da semana. Se o valor inserido não corresponder a nenhum dos casos, o bloco default é executado, retornando "Número inválido". 

switch (número) {
      case 1: resultado = "Domingo"; break;
      case 2: resultado = "Segunda-feira"; break;
      // ...
      default: resultado = "Número inválido";
  }
